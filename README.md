# 📊 Customer Service & Churn Analysis Dashboard

## 📌 Project Overview

This project is an interactive **Power BI Dashboard** developed to analyze customer subscriptions, services, billing, and support activities. The dashboard provides valuable business insights through interactive visualizations and KPIs, enabling stakeholders to monitor customer behavior, service usage, and support performance effectively.

The dataset was cleaned and prepared before being imported into Power BI to ensure accurate analysis and reliable reporting.

---

# 🎯 Objectives

* Analyze customer subscription patterns.
* Monitor customer service usage.
* Evaluate support ticket distribution.
* Track revenue-related metrics.
* Create an interactive dashboard for business decision-making.

---

# 🛠️ Tools & Technologies

* Microsoft Power BI
* Microsoft Excel

---

# 🧹 Data Preparation

The dataset was cleaned and transformed before visualization to improve data quality.

The preprocessing process included:

* Removing duplicate records
* Handling missing values
* Correcting inconsistent data
* Standardizing categorical values
* Formatting data types
* Renaming columns for better readability

---

# 📈 Key Insights

The dashboard helps answer important business questions, including:

* Which contract type has the highest customer base?
* Which internet service is most commonly used?
* What is the distribution of customers by gender?
* How many customers use value-added services such as Online Security and Streaming TV?
* How many technical and administrative support tickets have been generated?
* What is the total revenue generated from customers?
* How are customers distributed across different tenure groups?

# 🚀 Getting Started

1. Download or clone this repository.
2. Open the `.pbix` file using **Microsoft Power BI Desktop**.
3. If prompted, reconnect the dataset.
4. Refresh the data.
5. Explore the dashboard using the available slicers and filters.
